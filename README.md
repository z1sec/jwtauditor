# 🔐 JWTAuditor - 汉化修改版

# 主要功能
- 高级安全分析 - 自动漏洞检测及详细说明
- 秘密暴力破解 - 测试常用秘密和自定义单词列表
- JWT 编辑器 - 修改支持各种签名算法的令牌
- JWT 生成器 - 通过 RSA 密钥生成从零创建令牌
- 高级攻击平台 - 7个专用攻击模块，用于全面测试
- 全面文档 - 详细学习JWT安全
- 100%客户端 - 你的令牌永远不会离开浏览器

# 修改内容
1、原版是英文的，本人给汉化了；
2、有一些服务端的请求，我给去除掉了，保障离线环境完全可用；
3、把项目本身的10w+的字典和tscan的10w+字典合并去重，生成一个12w+字典。
